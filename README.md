# Undangan Pernikahan Modern

Undangan pernikahan digital yang modern dan interaktif dengan tema elegan dan efek visual yang menarik.

## Fitur Utama

### 🎨 **Desain Modern**
- Tema gradient yang elegan dengan warna pink dan ungu
- Typography yang indah menggunakan font Dancing Script dan Poppins
- Layout responsif yang sempurna di semua perangkat
- Animasi smooth dan transisi yang halus

### 📸 **Galeri Foto**
- Integrasi otomatis dengan foto-foto dari folder
- Lightbox effect untuk melihat foto dalam ukuran penuh
- Hover effects yang menarik
- Grid layout yang responsif

### 🎵 **Musik Latar**
- Music player yang elegan di pojok kanan bawah
- Kontrol play/pause dan stop
- Auto-play dengan fallback untuk browser yang memblokir autoplay
- Visual feedback yang menarik

### ✨ **Efek Keren**
- Floating hearts animation di background
- Parallax scrolling effect
- Ripple effect pada tombol
- Loading screen yang menarik
- Smooth scroll navigation
- Typing effect pada judul
- Intersection Observer untuk animasi saat scroll

### 📱 **Responsive Design**
- Mobile-first approach
- Hamburger menu untuk mobile
- Touch-friendly interface
- Optimized untuk semua ukuran layar

### 📝 **Form RSVP**
- Form konfirmasi kehadiran yang lengkap
- Validasi input
- Notification system
- Styling yang konsisten dengan tema

## Cara Menggunakan

1. **Buka file `index.html`** di browser modern
2. **Ganti foto** dengan foto Anda sendiri (ganti nama file di HTML)
3. **Edit informasi** seperti nama mempelai, tanggal, dan lokasi
4. **Tambahkan musik** dengan mengganti URL di elemen audio
5. **Customize warna** dan styling sesuai preferensi

## Struktur File

```
├── index.html          # File HTML utama
├── style.css           # Styling dan animasi
├── script.js           # JavaScript dan interaktivitas
├── README.md           # Dokumentasi
└── [foto-foto]         # Foto-foto undangan
```

## Kustomisasi

### Mengganti Foto
Edit bagian berikut di `index.html`:
```html
<img src="nama-foto-anda.jpg" alt="Deskripsi">
```

### Mengganti Musik
Edit bagian audio di `index.html`:
```html
<audio id="backgroundMusic" loop>
    <source src="path-ke-musik-anda.mp3" type="audio/mpeg">
</audio>
```

### Mengganti Informasi
Edit bagian couple names, tanggal, dan lokasi di `index.html`:
```html
<h2 class="groom-name">Nama Mempelai Pria & Wanita</h2>
<p class="hero-date">Tanggal Pernikahan</p>
```

### Mengganti Warna Tema
Edit CSS variables di `style.css`:
```css
:root {
    --primary-color: #e91e63;
    --secondary-color: #667eea;
    --accent-color: #764ba2;
}
```

## Tips Optimasi

1. **Kompres foto** sebelum digunakan untuk loading yang lebih cepat
2. **Gunakan format WebP** untuk foto untuk performa terbaik
3. **Optimize musik** dengan format MP3 atau OGG
4. **Test di berbagai device** untuk memastikan responsivitas
